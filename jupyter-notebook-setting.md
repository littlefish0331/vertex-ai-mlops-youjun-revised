# Jupyter Notebook Setting Tips

因為 workbench 的字體大小很不友善，所以建議要修改兩個地方。  
修改位置 `Seetings > Advanced Settings Editor`

- Terminal
  - Font size: 18
- Theme
  - code-font-size: 18px
  - content-font-size1: 18px
  - ui-font-size1: 18px  

設定錯惹也不用怕，有 `Restore to Defaluts` 可以使用。  
此設定在 Workbench(Notebook) 重新 Reset 也會存在。
